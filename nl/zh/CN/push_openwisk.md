# 与 Cloud Functions 集成
{: #cloud-functions-integration}

IBM Cloud functions 中的 {{site.data.keyword.mobilepushshort}} 软件包提供一套与 IBM {{site.data.keyword.mobilepushshort}} 交互的操作。通过这些操作，可以发送消息，或者创建、更新或删除 Webhook。
{: shortdesc}

要使用 IBM Cloud functions 中的最新推送通知服务功能，请通过 CLI 安装推送服务软件包。有关 {{site.data.keyword.mobilepushshort}} 软件包的更多信息，请参阅
[可安装的软件包](https://cloud.ibm.com/docs/openwhisk?topic=cloud-functions-push-notifications-package#push-notifications-package)。
