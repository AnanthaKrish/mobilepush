# 릴리스 정보
다음 새 기능 및 서비스 변경사항이 사용 가능합니다. 이들 변경사항은 기존 코드에 문제를 발생시키지 않습니다. 

## 2018년 3월 1일

- **메시지 전달 상태 추적**

    서비스에 제출된 모든 알림의 전달 상태를 볼 수 있습니다.  

- **알림 매개변수화**

    변수를 작성하고 알림 템플리트에서 이를 호출하여 사용자 정의 알림을 매개변수화하고 전송할 수 있습니다.
